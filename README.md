# sexecam.pro
SexeCam.pro est la première plateforme francophone qui combine caméras en direct, avatars IA interactifs, et dialogues intelligents 24h/24. Choisissez entre de vrais modèles vérifiés ou des personnalités IA entièrement générées. Tout est en direct, instantané et personnalisé selon vos envies. SexeCam réinvente l’expérience live hot!

## Exemple d'authentification (Firebase)

Ce dépôt inclut un exemple minimal d'intégration Firebase Authentication côté client (dossier `firebase-auth/`) :

- `firebase-auth/index.html` — pages d'inscription, connexion, récupération de mot de passe et tableau de bord minimal.
- `firebase-auth/app.js` — logique JS qui utilise Firebase Auth (email/password) et gère l'envoi de la vérification d'email.
- `firebase-auth/README.md` — instructions pour créer un projet Firebase et récupérer la configuration.

Suivre les instructions dans `firebase-auth/README.md` pour créer la Web App Firebase et copier la configuration (objet `firebaseConfig`) dans `firebase-auth/app.js`.
